- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- 모바일 에서 가능한 반응형 웹 구현
- 시맨틱 태그 구조로 생성
- body

[레이아웃 구조]

1. 상단 Header
   - 좌측: 보험사명 표시
   - 우측: 타이머(60:00), 연장 버튼, 사용자명, 로그아웃 아이콘
   - height 60px 고정

2. 좌측 Sidebar
   - width 240px
   - 메뉴: 상품자료 / 니즈자료
   - 하단: 관리자 설정
   - 세로 전체 고정
   - active / hover 상태 구현

3. 메인 Content 영역
   A. 상품 검색 영역
      - Select 박스 4개 (가로 정렬)
      - 검색 버튼
      - 정렬은 flex
      - 적절한 간격 유지


      B. select 박스
      - 4개
      - 첫번째 select 박스는 "전체"로 고정
      - 두번째 select 박스는 "전체"로 고정
      - 세번째 select 박스는 "전체"로 고정
      - 네번째 select 박스는 "전체"로 고정
      - 코드값은 아래 참고


      1번 값은 
      생명보험, 손해보험
      2번 값은 
      삼성생명보험, 한화생명, 교보생명
      3번 값은
      2번의 선택 값에 따라 달라집니다
      2번 값이 삼성생명인 경우
      삼성생명 종신보험
      2번 값이 한화생명인 경우 
      한화생명 종신보험
      
      4번 값은 
      상품요약서, 사업방법서, 약관
      
      값의 코드명들은 임의로 지정해주세요



   B. 카드 리스트 영역
      - 3열 Grid 구조
      - 카드 구성:
         - 상단 이미지 (비율 유지)
         - 하단 다운로드 버튼 + 아이콘
      - 카드 hover 효과 적용
      - border-radius 12px
      - box-shadow 적용

[UI 디자인 가이드]

- primary 컬러는 블루 계열
- 버튼은 radius 8px
- 폰트는 Noto Sans KR
- 기본 폰트 크기 14px
- spacing은 8px 기준

[기능 요구사항]

- 검색 버튼 클릭 이벤트 생성
- 타이머는 setInterval로 구현
- 다운로드 버튼 클릭 시 console.log 출력

[반응형 조건]

- 1024px 이하 → 2열 Grid
- 768px 이하 → 1열 Grid
- 모바일에서 Sidebar는 숨김 처리

[코드 작성 조건]

- Semantic HTML 사용
- class 네이밍은 BEM 방식
- CSS는 별도 style 태그에 작성
- 코드 전체를 하나의 파일로 작성
- 주석 충분히 작성

완성된 전체 코드를 출력해주세요.
