

## 모든 html과 css파일에 아래 컨셉을 적용해서 코딩할것

모든 html과 css파일에 아래 디자인 컨셉을 적용해서 개발할것
비지니스 로직은 제외하고 디자인과 퍼블리싱만 구현할것


## 2. 🎨 디자인 컨셉

### ✔ 전체 컨셉

* 안정감 있는 **Blue 계열 관리자 UI**
* 정보 전달 중심의 **클린 & 기능 중심 디자인**
* 과한 디자인 요소 배제 (Flat Design)
* 전체적으로 깔끔한 디자인을 유지할것
* 요소에 호버 미적용
* 모든 테두리 Radius는 0으로 할것(미적용)


## 3. 🎨 컬러 시스템

### ✔ Primary Color

* Main Blue: `#2f6fed`
* Sub Blue: `#4da3ff`


### ✔ Background

* 전체 배경: `#f5f7fa`
* 컨텐츠 영역: `#ffffff`

### ✔ Border

* 기본 Border: `#dcdcdc`
* Table Border: `#e5e5e5`

### ✔ Text

* 기본 텍스트: `#333333`
* 보조 텍스트: `#666666`
* 비활성 텍스트: `#999999`
