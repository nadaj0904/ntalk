
IMPORTANT

로컬 환경(IPv4)에서 Supabase에 연결하기 위해 아래 절차를 따라 Connection String을 복사하여 답변으로 알려주시기 바랍니다:

Supabase 대시보드에 로그인합니다.
nproject2 프로젝트 선택 후 화면 왼쪽 메뉴에서 Project Settings (톱니바퀴) -> Database로 이동합니다.
화면 아래쪽 Connection string 항목으로 이동하여 Connection pooler 탭을 클릭합니다.
JDBC 탭이 있다면 선택한 뒤(또는 일반 URI 복사), 해당 URL과 표시되는 Username을 모두 복사해서 저에게 알려주세요.
예시: URL: jdbc:postgresql://......pooler.supabase.com:6543/postgres Username: postgres.abcdefg
